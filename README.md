## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository with Typeorm support along with Migrations.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# generate migrations
$ npm run migration:generate migrationName

# run migrations
$ npm run migrate

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
